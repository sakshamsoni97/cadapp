\subsection{Detailed Description}
Contains \hyperlink{class_projection}{Projection} and \hyperlink{class_object3_d}{Object3D} class definitions. 

This is the interface to the library developed for the C\+AD application which implements all the required algorithms. Libraries used\+:
\begin{DoxyItemize}
\item stl\+::list
\item G\+L\+EW\+: Open\+GL Extension Wrangler Library
\item G\+L\+UT\+: Open\+GL Utility Toolkit
\item Cereal\+: Library to serialize objects and store them in binary format
\end{DoxyItemize}

Open\+GL libraries are used to render orthographic views and the 3D object
